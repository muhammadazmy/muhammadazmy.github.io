Cara Gampang Upload/Deploy Project ke Github

Autor : Muhammad Azmi

Dasar Git Jadi, sebenarnya apa yang dimaksud dengan Git? Ini adalah bagian penting untuk dipahami, karena jika anda memahami apa itu Git dan cara kerjanya, maka dapat dipastikan anda dapat menggunakan Git secara efektif dengan mudah. Selama mempelajari Git, cobalah untuk melupakan VCS lain yang mungkin telah anda kenal sebelumnya, misalnya Subversion dan Perforce. Git sangat berbeda dengan sistem-sistem tersebut dalam hal menyimpan dan memperlakukan informasi yang digunakan, walaupun antar-muka penggunanya hampir mirip. Dengan memahami perbedaan tersebut diharapkan dapat membantu anda menghindari kebingungan saat menggunakan Git.

Bahasa Sederhananya, tempat untuk mendokumentasikan project, kamu juga bisa bekerja secara bersamaan jadi project kamu bisa cepat selesai walaupun satu tim saling berjauhan.

Langsung saja pada tutorialnya:

  1. Buat akun pada https://github.com/
  2. Setelah selesai, lalu download Git, download disini https://git-scm.com/downloads, pilih sesuai OS masing- masing
  3. Instal seperti biasa
  4. Lalu login ke akun github dan buat repository baru, klik link

    5. lakukan seperti gambar diatas: Repository name: (Diisi dengan nama nama_web.github.io; contoh: portofolioku.github.io) Desciption: (Diisi deskripsi repositorynya (tidak wajib)) lalu set Public, Private, default Public, lalu klik tombol berwarna hijau Create Repository.

  5. Setelah Create Repository muncul halaman seperti ini:
  6. Setelah itu buka project yang akan dimasukkan ke Github, caranya klik kanan pada folder project dan pilih Git Bash Here

    7. dan muncul command line atau Terminal

  7. Ketikkan perintah seperti di step 5:
    8.                                 
                                    echo "# portofolioku.github.io" >> README.md
                                    git init
                                    git add README.md
                                    git commit -m "first commit"
                                    git branch -M main
                                    git remote add origin https://github.com/muhammadazmy/portofolioku.github.io.git
                                    git push -u origin main
      Penjelasan:
    1. git add = untuk memiliih file yang akan dimasukan ke github (kalau pilih semua: “ git add . “ )
    2. git commit = untuk memberi commit/pesan git remote = alamat untuk memasukan ke github
    3. git push = untuk ngepush semuanya ke github klik enter dan lihat project kamu sudah terupload ke Github
  8. Selamat Project kamu sudah masuk ke Github!, dan tunggu beberapa menit sehingga website siap di gunakan
